## [Velvet Web](https://github.com/VelvetProjects/VelvetWeb)
**Velvet web** is a browser that will **help users focus** with [Apple's Focus feature](https://support.apple.com/en-au/guide/iphone/iphd6288a67f/ios). Please note that it will be **inspired by apple's feature**, not a direct copy. 

### Features
- Basic Browser features
   - Search bar
   - Back/Forward buttons
   - User accounts
- Focus mode
   - Do not disturb -- silences web notifications and closes other tabs)
   
### Why?
Ever since JaydenRidley4 made [LiteIE](https://github.com/jaydenridley4/liteie)-- Internet explorer but simplified. We got quite inspired and decided to start development to **Velvet Web**.
